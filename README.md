# BMS Monitor

A lightweight macOS menu bar application for monitoring Battery Management System (BMS) data in real time using Firebase Realtime Database.

## Features

- Real-time battery monitoring
- Menu bar dashboard
- Native macOS design
- Firebase Realtime Database support
- Configurable GoogleService-Info.plist
- Custom database path
- Notifications
- Particle animation

## Setup

1. Clone the repository
2. Open the project in Xcode
3. Create your Firebase project
4. Download GoogleService-Info.plist
5. Configure it from Settings
6. Run
